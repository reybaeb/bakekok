auto_awesome Cara Menggunakan

1. Pilih Jenis File: Pilih Gambar, Dokumen (PDF), atau Arsip (ZIP) pada panel atas.

2. Upload File: Seret file ke area kotak atau klik area tersebut.

3. Opsi Agresif: Geser tombol "Mode Agresif" jika butuh kompresi maksimal (kualitas turun).

4. Eksekusi: Tekan tombol hitam besar "MULAI KOMPRESI".

warning Kenapa ukuran file bertambah?

File yang sudah optimal (seperti JPG terkompresi) jika dikompres lagi justru akan menambah metadata. Ini adalah perilaku normal algoritma.

tune Detail Teknis

Kompresi PDF

  • Standar: Pembersihan struktur file.
  • Agresif: *Re-sampling* seluruh gambar dalam PDF ke resolusi lebih rendah.

Kompresi ZIP

  • Standar: Re-pack Deflate (Level 9).
  • Agresif: Ekstrak isi -> Kompres Gambar -> Re-zip.
